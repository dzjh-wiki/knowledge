# 经典物理方程式

----

## 1. 直线运动
### 1.1 匀变速直线运动
  * 位移：$$s = v_0t + \frac{1}{2}at^2$$

#### 1.2 自落体体运动
  * 下落高度：$$h = \frac{1}{2}gt^2$$

#### 1.3 竖直上抛运动
  * 位移：$$h =v_0t - \frac{1}{2}gt^2$$

## 2. 相互作用
### 2.1 常见的力
  * 重力：$$G = mg$$
  * 胡克定律：$$F = kx（方向沿恢复形变的方向）$$
  * 滑动摩擦力：$$F = \mu F_N（方与物体相对运动方向相反）$$
